## 1. 产品概述

一款融合知识答题与文明建造的互动科普游戏，玩家通过答题获取资源，建造各时代地标建筑，体验人类文明演进的壮丽史诗。

- 核心目标：以轻松有趣的方式科普人类文明发展史，让玩家在游戏中感受从原始部落到现代都市的文明跃迁
- 目标用户：对历史、科普、休闲游戏感兴趣的玩家，年龄8-80岁
- 产品价值：寓教于乐，用5分钟上手的轻量游戏体验，

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|---------|---------|
| 玩家 | 无需注册，直接游玩 | 完整游戏体验 |

### 2.2 功能模块

1. **主界面**：游戏标题、时代时间轴、资源显示、开始/重置功能
2. **答题系统**：随机题目展示、选项交互、反馈动画、知识解析
3. **建造系统**：地标建筑展示、建造进度、建造动画
4. **时代演进系统**：时代切换、视觉风格变化、背景音乐切换
5. **成就系统**：里程碑解锁、成就展示

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|-----------|
| 主游戏界面 | 时代时间轴 | 横向展示6个时代节点（原始部落→青铜时代→农业文明→工业革命→现代都市→未来展望），当前时代高亮显示 |
| 主游戏界面 | 资源面板 | 显示当前知识点数、建筑数量、文明等级 |
| 主游戏界面 | 文明场景展示区 | 根据当前时代动态渲染的场景（带时代主题背景、地标建筑、建造动画 |
| 主游戏界面 | 答题弹窗 | 弹出式题目卡片，4选1，答题后展示知识点解析 |
| 主游戏界面 | 建造面板 | 展示当前时代可建造的地标列表，消耗知识点数进行建造 |
| 主游戏界面 | 成就通知 | 新建造完成/时代解锁等事件的弹出通知 |

## 3. 核心流程

玩家进入游戏 → 从原始部落时代开始 → 通过答题获取知识点数 → 消耗知识点数建造地标建筑 → 所有地标建造完成后解锁新时代 → 循环直到完成全部时代 → 达成文明史诗成就。

```mermaid
flowchart TD
    A["玩家进入游戏"] --> B["展示游戏主界面"]
    B --> C["选择答题获取知识点"]
    C --> D{"答题是否正确"}
    D -->|是| E["获得知识点数 +解析展示"]
    D -->|否| F["展示正确答案解析"]
    E --> G["返回主界面"]
    F --> G
    G --> H["选择建造地标"]
    H --> I{"知识点数是否足够"}
    I -->|是| J["建造地标 +动画"]
    I -->|否| K["提示知识点数不足"]
    J --> L{"当前时代地标是否全部建成"]
    L -->|是| M["解锁新时代 +视觉切换"]
    M --> N{"是否完成全部时代"}
    N -->|是| O["达成文明史诗成就"]
    N -->|否| C
    L -->|否| C
    K --> C
```

## 4. 用户界面设计

### 4.1 设计风格

**整体风格：史诗感 + 精致复古 + 大开大合的人类简史风格

- **主色调**：
  - 原始部落时代：土黄赭石色系（#8B6914, #CD853F, #DEB887）
  - 青铜时代：青铜绿金色系（#B8860B, #DAA520, #6B8E23）
  - 农业文明：田园绿褐色系（#556B2F, #8FBC8F, #D2691E）
  - 工业革命：钢铁蓝灰色系（#708090, #4682B4, #2F4F4F）
  - 现代都市：霓虹蓝紫色系（#4169E1, #9370DB, #00CED1）
  - 未来展望：赛博粉青色系（#00FFFF, #FF69B4, #7FFFD4）

- **按钮风格**：圆角矩形，带有时代特色边框，悬浮时上浮+发光效果
- **字体**：标题使用 Cinzel Decorative（古典史诗感），正文使用 Cormorant Garamond（优雅易读）
- **布局风格**：卡片式布局，时间轴横向贯穿顶部，场景区域居中展示
- **图标/emoji**：使用像素艺术风格图标，每个时代有专属 emoji 标识

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|---------|---------|---------|
| 主游戏界面 | 时代时间轴 | 横向流动设计，节点发光指示当前时代，过渡动画平滑 |
| 主游戏界面 | 文明场景展示区 | 全屏沉浸式场景，SVG/Canvas绘制时代特色场景，地标建筑逐个浮现动画 |
| 主游戏界面 | 答题弹窗 | 居中卡片式，题目突出，选项悬浮放大，正确绿色脉动，错误红色抖动 |
| 主游戏界面 | 建造面板 | 右侧抽屉式或底部横排，建筑图标+名称+消耗点数 |
| 主游戏界面 | 资源面板 | 顶部状态栏，数字跳动动画 |

### 4.3 响应式

桌面优先设计，适配平板和移动端自适应布局，时间轴在移动端改为纵向滚动。

### 4.4 动画与视觉特效

- 场景转换：时代切换时全屏渐变色过渡+粒子特效
- 建筑建造：从地基到建成的生长动画+粒子爆发
- 答题反馈：正确答案绿色光晕扩散，错误答案红色抖动
- 背景：每个时代有独特的动态背景元素（如原始时代的篝火、现代都市的霓虹等）
