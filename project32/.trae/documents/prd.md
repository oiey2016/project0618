## 1. 产品概述

「知识探险队」是一款科普答题冒险游戏，玩家扮演探险家在神秘大陆探险，途中遭遇各种关卡挑战，通过回答科学知识题目获取金币，用金币购买装备提升能力，最终通关整个冒险。

- **核心玩法**：冒险探索 + 知识答题 + 装备成长
- **目标用户**：8-16岁青少年及所有喜欢科普知识的玩家
- **产品价值**：寓教于乐，在游戏中学习科学知识

## 2. 核心功能

### 2.1 功能模块

1. **冒险地图**：关卡地图、关卡节点、进度保存
2. **答题系统**：题目展示、选项选择、正误判定、奖励发放
3. **金币系统**：金币获取、金币消费、金币显示
4. **商店系统**：装备购买、装备展示、价格管理
5. **装备系统**：装备穿戴、属性加成、背包管理
6. **角色系统**：角色属性、等级经验、能力成长

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 主菜单 | 游戏标题、开始按钮、设置按钮 | 进入游戏、音效开关 |
| 冒险地图 | 关卡节点、路径展示、进度指示 | 选择关卡、查看进度 |
| 答题界面 | 题目区、选项区、反馈区 | 展示题目、选择答案、显示结果 |
| 商店界面 | 商品列表、购买按钮、金币显示 | 浏览装备、购买装备 |
| 背包界面 | 装备列表、穿戴按钮、属性面板 | 查看装备、穿戴装备、查看属性 |
| 胜利界面 | 奖励展示、继续按钮 | 显示通关奖励、返回地图 |

## 4. 用户界面设计

### 4.1 设计风格

- **整体风格**：奇幻冒险卡通风格，色彩明亮丰富
- **主色调**：深蓝色（#1e3a5f）作为背景主色，金色（#ffd700）作为强调色
- **辅助色**：翡翠绿（#10b981）成功色、橙红（#ef4444）失败色、紫色（#8b5cf6）稀有度
- **按钮风格**：圆角立体按钮，带有渐变和阴影，悬停有弹跳效果
- **字体**：使用圆润可爱的标题字体，清晰易读的正文字体
- **布局风格**：卡片式布局，信息分区明确
- **图标风格**：卡通风格 emoji + lucide图标

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|---------|---------|--------|
| 主菜单 | 标题区域 | 大标题渐变文字、发光效果、装饰图案 |
| 主菜单 | 按钮区域 | 开始游戏按钮、设置按钮、音效开关 |
| 冒险地图 | 顶部状态栏 | 金币显示、角色头像、等级 |
| 冒险地图 | 地图区域 | 关卡节点、路径连线、背景装饰 |
| 答题界面 | 题目卡片 | 题目标题、题目内容、进度条 |
| 答题界面 | 选项区域 | 四个选项按钮、选中效果、正确错误反馈 |
| 答题界面 | 底部信息 | 当前金币、题目序号、倒计时（可选） |
| 商店界面 | 顶部栏 | 返回按钮、金币显示 |
| 商店界面 | 商品列表 | 装备卡片、价格标签、购买按钮 |
| 背包界面 | 角色面板 | 角色形象、属性数值 |
| 背包界面 | 装备列表 | 已拥有装备、装备操作按钮 |

### 4.3 响应式

- 桌面端优先设计，适配平板和移动端
- 移动端采用垂直布局，按钮增大触控区域
- 支持横竖屏自适应

### 4.4 动效设计

- 关卡解锁时的庆祝动画
- 答对题目时的金币飞入动画
- 按钮悬停的微弹跳效果
- 页面切换的平滑过渡
- 装备穿戴的闪光效果
